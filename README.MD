# Wedding Invitation - Django Project

## Description

This project is a Django-based web application that enables sending online wedding invitations. Users can fill out an RSVP form to confirm their attendance and view images in an interactive slider.

## Features

- **Home page with the invitation**  
- **RSVP Form** (Name, Email, Message, Attendance Checkbox)  
- **Interactive image slider with 3 images**  
- **Django Admin panel** for managing guests  
- **API to retrieve and submit RSVPs**
